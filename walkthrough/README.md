# Walkthrough Files

Store agent-written gameplay notes in this directory.

Guidelines:

- Read relevant walkthrough files here before acting if they exist.
- Prefer server-specific files such as `walkthrough/aardwolf.md` or `walkthrough/custom-gbk-mud.md`.
- Add new markdown files here when a server or task needs a dedicated walkthrough.
- Update these files while playing with important discoveries, safe routes, login quirks, and workflow fixes.
- Keep secrets out of walkthrough files. Credentials belong in `config/local.secrets.json`.
- Keep notes concise and operational so later agents can continue quickly.
